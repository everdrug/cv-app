Nginx Tutorial (1)

Deploying a simple static web server using Nginx with Docker

Install Docker

Need docker installed on the working machine. There are a lot of documents/tutorials available online explaining how to install and user docker. See docker document for example.

Nginx Configuration File

Nginx has a master process and multiple worker processes. Master process maintains the worker processes based on the provided configuration file, and worker processes process the requests. Therefore, we need to provide a configuration file.

Web Site

Nginx can be used for different purposes, such as web server, load balancer, api gateway, reverse proxy and so on. In this first tutorial, we demonstrate how to deploy a static web server using Nginx and docker. Therefore, we need some web pages.

Dockerfile

As we deploy our Nginx web server using docker, we need a dockerfile to configure how to build our docker image.
